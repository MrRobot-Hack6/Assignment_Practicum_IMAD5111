# Camp Commander

## About

Camp Commander is a simple Inventory management system for when you go camping, Adding items, their quantity and your own comments about the item.

It also allows the user to categorize them. e.g A tent would be catergorized as "Shelter".

## Programming Language
* Kotlin

## Camp Commander has 3 Screens:
* Splash Screen (Welcome Screen)
* Main Screen (Where items are added, comments are made and a total of items will display)
* Detailed view screen (This is the screen where all added items will be stored until the user clears the list.)
  

## The following categories are in the app:

* Shelter
* Food
* Safety
* Cooking
* First Aid
* Cloting
